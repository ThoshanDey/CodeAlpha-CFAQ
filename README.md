#FAQ Chatbot
A simple FAQ Chatbot built using Python that can answer predefined questions using basic Natural Language Processing (NLP) techniques.


## Features
- Answers frequently asked questions
- Rule-based response system
- Easy to modify and expand FAQ dataset
- Lightweight and fast execution
- Beginner-friendly Python project


## Technologies Used
- Python 3
- NLTK (Natural Language Toolkit)
- Basic file handling / dictionary-based logic
